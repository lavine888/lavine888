## ♓ 关于我

我是一名数学与计算机科学方向的学习者，长期关注 AI 产品、量化建模与数据系统。

我的核心方向，是把模糊想法推进成可运行的系统，包括 AI 应用、产品原型、数据管道，以及可测试的量化策略。
